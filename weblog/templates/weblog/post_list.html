Django Tutorial 1: Django 템플릿 생성

1. 현재 디렉토리에 장고 템플릿 생성
> django-admin startproject mysite .

2. 웹사이트 설정 변경
settings.py
TIME_ZONE = 'Asia/Seoul'
STATIC_ROOT = os.path.join(BASE_DIR, 'static')

3. DB 이관
> python manage.py migrate

4. 웹서버 기동
> python manage.py runserver
